安装步骤

下载或Clone该仓库

将CSS/images/js文件夹上传到本地服务器，如果你是Docker搭建的New Api 你就需要将这些文件做成直连进行使用。

打开11x.html文件全部粘贴或者复制到New APi 的系统设置首页设置文件中
